# Übung - Anlegen Verzeichnisse und Scripte 

## Walkthrough 

```
1. Im Heimatverzeichnis eine Verzeichnisstruktur erstellen 
   fotos/2020/januar 

2. In diesem Verzeichnis eine neue Datei anlegen mit dem Inhalt 
   Das sind meine Fotos 
   
3. in fotos/2020 ein Verzeichnis fehler anlegen 

4. dann verzeichnis fehler wieder löschen (rmdir, oder option -r )

5. Ein Script anlegen, mit dem Inhalt
   #!/bin/bash 
   echo "hallo ich bin's"
   
   anlegen als script.sh in einem verzeichnis, so dass es im Pfad liegt
   und als script.sh ausgeführt werden kann. 
   
```
